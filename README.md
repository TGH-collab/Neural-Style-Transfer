Neural Style Transfer Project
Overview

This project implements Neural Style Transfer using a pretrained VGG19 model to combine the content of one image with the style of another.

Features
Content and style image processing
Gram matrix for style representation
Loss optimisation using gradient descent
Multiple experiments with different style weights
Technologies Used
Python
PyTorch / TensorFlow
VGG19
Results

The system demonstrates how different style weights affect output:

Low weight → strong content
High weight → strong style
